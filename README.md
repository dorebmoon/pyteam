#python

本项目为 「1024编程实验室」Python 学习小组私有项目。

# doc 

学习文档目录，主要存放「大家」整理的通用的学习文档。

# notes

学习笔记，请每个人创建自己的私有文件夹，根据年月日创建笔记文档。

必须使用「markdown」的语法，不会的同学要自行安排学习。

# example

学习过程中，编写的例子，与大家分享。

- 请勿添加对别人无意义的例子代码
- 每个例子必须是独立的文件夹
- 文件夹命名规则：用户名_例子名称
- 每个例子必须包含 README.md 介绍例子的运行方法和学习目的

# project

等学习稍微成熟一点之后，编写的项目代码，置于此处。
